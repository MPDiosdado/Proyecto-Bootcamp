Antes de empezar, instalar las dependencias especificadas en el archivo requirements.txt y 
editar la variable DATABASE_URL en database.py con tus credenciales de PostgreSQL: postgresql://USUARIO:CONTRASEÑA@localhost:5432/ciudadai_db

* **`generador.py`**: Crea de forma aleatoria 500 solicitudes basadas en categorías como Seguridad, Limpieza y Tráfico.

* **`database.py`**: Describe el mapeo de estos datos a una tabla SQL mediante SQL
